# 竹荪

![](02.webp)

|属性|说明|
| ---- | ---- |
| 别称| 长裙竹荪|
| 属||
| 生长环境| 生长在竹林中腐朽的竹子及土壤上。|
| 外形特征| 形状略似网状干白蛇皮，它有深绿色的菌帽，雪白色的圆柱状的菌柄，粉红色的蛋形菌托，在菌柄顶端有一围细致洁白的网状裙从菌盖向下铺开，被人们称为“雪裙仙子”、“山珍之花”、“真菌之花”、“菌中皇后”。|
| 繁殖||
