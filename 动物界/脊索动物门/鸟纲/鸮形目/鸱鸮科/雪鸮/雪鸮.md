# 雪鸮

![](01.png)

|属性|说明|
| ---- | ---- |
| 别称| 雪枭，哈利波特的猫头鹰|
| 分布| 高纬度和高海拔的寒冷地区。|
| 寿命||
| 外形特征||
| 食性| 主要以北极地区常见的小型哺乳动物为食，主要包括旅鼠和幼岩雷鸟，食物匮乏时也会游荡到其他地域取食其他啮齿类动物、雉类、雁鸭类和雪兔等。不能被砂囊吸收或被排泄的东西以食丸的形式反吐出来。|
| 繁殖| 雪鸮终身为一夫一妻制，不过当食物极度充足时也可能有一夫多妻制的情况。|
